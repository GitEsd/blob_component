# Custom credential/connection example

## What is a custom credential/connection?

You can create custom connection objects to connect to external systems. To make those connections configurable you can also
create custom credentials that can be configured in the flows ui.

## What does this example demonstrate?

This example demonstrates how to create a custom connection and credential object.

## How to build the example

For all commands, make sure you are in the root folder of this example component (the same folder this file is located in).

To build the project, run `./gradlew build`.

To build the project without running unit tests, use `./gradlew build -x test`.

To run only the unit tests, execute `./gradlew test`.
